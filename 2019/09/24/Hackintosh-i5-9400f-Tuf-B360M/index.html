规格 详细信息

哪些不能在Clover正常工作

由于没有购置 蓝牙无线网卡 (例如免驱的 BCM943602CS ) ,所以无法测试蓝牙与airdrop是否正常工作,理论上打上驱动是没问题的

哪些能在Clover正常工作

暂时用了这么久都能正常工作使用(什么傻* markdown 格式显示不出来…反正 x 就是成功能用的)

  • [x] 正常上网 (由于公司使用的是 USB 外置网卡EP-AC1606 （8812AU) 需要安装驱动!, 自己测试过 使用网线也是能上网的,所以驱动正常！)

  • [x] 能休眠,能唤醒,唤醒后无异样(如花屏,断网等)

  • [x] 声卡能播放声音(并不是使用万能声卡驱动)

  • [x] 能正常关机重启

  • [x] 能屏幕保护(有点低要求啊….)

  • [x] DP口,HDMI口能用

  • [ ] 双屏输出(未测试,待后续更新)

  • [x] USB口都能用

硬件参数参考

这一部分介绍下买的硬件具体样子与参数,因为同一个东西也会翻车哦 (例如 RX580 2048SP 是无法免驱的…)

CPU:

CPU来说只要不是买 AMD 就行了(没黑成分..毕竟白苹果没有用过AMD啊…不是不能上,又要折腾一番何必呢…)

另外就是注意买的CPU支持你的板子BIOS与针脚，比如你买的CPU很新，板子虽然支持这一代CPU，但是需要刷BIOS才认得出这种CPU，就不要搞了,BIOS 版本太新也会坑(后面会说一下我遇到的折腾的坑),上一下我的 CPU图吧

安装

制作启动U盘那些就不说了，自行Google,

这次安装的系统是 10.14.5, 使用的是 Clover 4920

Clover传送门

由于有过2次黑苹果安装经验(Z270 n卡一次, a卡一次),以为主流板子都一个套路

就用了Z270 当时的UEFIDriver与Kext 来安装，plist也按那套

然后也按照网上 B 板子的主板参数调整了一下（后面会说到）

能正常进入安装界面,并选择 APFS 格式的抹盘，结果装到最后2分钟卡了.

按照经验判断，认为是内存的驱动有问题,替换掉 AptioMemoryFix-64 驱动，改成了 osxaptiofix2drv-free2000 、osxaptiofix3drv 、osxaptiofix2drv 、osxaptiofixdrv 任意一个也不行..(重启或者卡死)…已经开始怀疑人生了

换个思路，会不会板子BIOS太新呢? 在官网下载BIOS更新

我是传送门

然后看了下板子的BIOS 是 2418，然后一步步降级到 2401 ，Bios不给再降了. 也是上面问题. 还以为这次翻车了.

跟朋友交流下经验.他的板子是 ASUS TUF Z390 也是卡了最后2分钟!!!!!! 难道是板子问题！！？？

他说最后2分钟然他卡半小时，然后重启就可以进入下一步安装界面了，然后再卡1小时再能进下一步

王德发，太TM坑了吧…但是都翻车了何不再试下..

结果能去到第二步,卡了一小时后重启也是一样没有下一步…

无奈下….某东下单一块技嘉板子….有图有真相

Google 百度几乎搜烂也没任何结果…此时我只能怪责板子的锅了..

几经搜索….好像看到曙光….
